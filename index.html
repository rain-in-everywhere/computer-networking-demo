计算机网络 · Serverless 实践

基于 Cloudflare Workers 搭建的 Serverless 网站。 从零实现 Web 服务，深入理解 CDN、DNS、HTTP 缓存、TLS 加密等核心计网概念。

📖 关于本项目

本项目是计算机网络课程大作业，使用 Cloudflare Workers 构建了一个完整的 Serverless 网站。 不同于传统的服务器部署，Worker 运行在 Cloudflare 全球 330+ 个边缘节点上， 就近响应用户请求，天然具备 CDN 加速和高可用能力。

网站本身就是一个计网实验场：每个页面、每次 API 调用都在展示网络协议的工作原理。 打开浏览器的 DevTools → Network 面板，你将看到 HTTP 请求/响应的完整细节。

⚡ HTTP 缓存策略

涉及概念：DNS · DoH · 记录类型 · 域名层次

涉及概念：FaaS · V8 Isolate · 冷启动 · 边缘执行

🛠 技术栈与架构

边缘计算

Cloudflare Workers 运行在 V8 Isolate 中， 无冷启动容器开销，全球 330+ 边缘节点就近响应。

Serverless

无需管理服务器、无需运维。代码部署到边缘网络后自动扩展， 按请求量计费（免费额度 10万次/天）。

HTTPS 全链路

Cloudflare 自动签发 SSL 证书，用户→边缘→源站全程 TLS 加密。 支持 HTTP/2 和 HTTP/3 (QUIC)。

零框架

纯 HTML/CSS/JS，零构建步骤。 每个 HTTP 请求都在你的掌控之中——这正是本课程的学习目标。
